Installing on Android

  1. Download the .apk for the version you want.
  2. Open the downloaded file. If prompted, allow your browser to install unknown apps (Settings → Apps → Special access).
  3. You need two phones: one becomes the Baby Unit, the other the Parent Unit. Install on both.
